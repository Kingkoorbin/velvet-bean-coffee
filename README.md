# Velvet Bean Coffee Co. — Modern Single-Page Website

A fully responsive, interactive coffee shop website built with **zero dependencies**.
Features dark/light mode, auto-playing carousel, tabbed menu system, lightbox gallery,
scroll animations, and a working contact form.

## Live Demo
[View Live Site](https://yourusername.github.io/velvet-bean-coffee)

## Features
- Dark/Light mode toggle with persistence
- 4-slide auto-playing hero carousel with progress bar
- 5 tabbed menu categories (24 items)
- Masonry gallery with lightbox viewer
- Scroll-triggered reveal animations
- Animated stat counters
- Working contact & newsletter forms
- Mobile-responsive hamburger menu
- Custom cursor glow effect (desktop)

## Tech Stack
- HTML5 semantic markup
- CSS3 (Grid, Flexbox, Custom Properties, Animations)
- Vanilla JavaScript (ES6+)
- Intersection Observer API
- LocalStorage API

## Performance
- Zero external JS frameworks
- Lazy-loaded images
- Minimal repaints with CSS transforms
